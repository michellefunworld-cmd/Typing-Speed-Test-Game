⌨️ Typing Speed Test Game

An interactive browser-based typing speed test built with HTML, CSS, and JavaScript. This project allows users to measure their typing speed and accuracy in real time while improving focus and consistency.

🚀 Features

Random sentence generation

Real-time countdown timer

Words Per Minute (WPM) calculation

Accuracy percentage tracking

Instant results display

Restart functionality

Responsive and clean design

🎮 How to Play

Open index.html in your browser.

Begin typing the displayed sentence.

Type as quickly and accurately as possible before the timer ends.

When the test finishes, your WPM and accuracy will be shown.

Click restart to try again and improve your score.

📊 How It Works

WPM is calculated using the standard formula:

(Total characters typed ÷ 5) ÷ Time in minutes

Accuracy is calculated by comparing correctly typed characters to total typed characters.
